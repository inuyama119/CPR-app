# 胸骨圧迫評価アプリ システム仕様書 (v3.40 安定版)

このドキュメントは、2026年5月時点での「最も安定し、精度が高い」と判断された v3.40 の全仕様を記録したものです。不具合発生時や、以前の仕様に戻したい際のマスターデータとして使用してください。

## 1. 圧迫計測ロジック (コアアルゴリズム)
手首の動きをベースとした、ノイズに強く感度の高い計測設定です。

| 項目 | 設定値 | 役割 |
| :--- | :--- | :--- |
| **判定ポイント** | 手首 (Wrist) | 左右でより可視性が高い方を自動選択。 |
| **判定しきい値** | `0.01` | 圧迫として認める最小の上下動幅。浅い圧迫も拾える感度。 |
| **不感時間 (Dead-time)** | `400ms` | 圧迫直後の「跳ね返り」を誤検知しないガード時間。150BPMまで対応。 |
| **スムージング (EMA)** | `0.4` | 映像のガタつきを抑え、数値を安定させるフィルタ係数。 |
| **BPM計算方式** | 指数移動平均 | 直近3回程度の傾向を重視しつつ滑らかに表示。 |

## 2. 姿勢評価基準
AIが「正しい姿勢」と判断するための数学的しきい値です。

*   **肘の角度**: **115度以上** で合格（腕が真っ直ぐ伸びている状態）。
*   **垂直のズレ (Tilt)**: 基準値から **0.15以内** で合格（肩の真下で押せている状態）。
*   **警告・中断ルール**:
    *   **警告**: 姿勢不良が **2秒** 継続で警告音とメッセージを表示。
    *   **中断**: 姿勢不良が **5秒** 継続で練習を強制終了。

## 3. 採点ロジック (リザルト画面)
ユーザーの達成感を重視した3段階/2段階評価です。

*   **リズム評価**:
    *   **◎ (達成)**: 平均BPMが **100～120** の範囲内。
    *   **△ (要練習)**: 上記範囲外。
*   **姿勢評価**:
    *   **◎**: 肘・ズレの両方が合格。
    *   **○**: どちらか一方が合格。
    *   **△**: 両方不合格。

## 4. UIデザインシステム
「清潔感・プロフェッショナル・緊迫感」を両立したカラーパレット。

| 要素 | カラーコード | 役割 |
| :--- | :--- | :--- |
| **Primary (Pink)** | `#FF4B72` | メインボタン、重要メッセージ。 |
| **Secondary (Blue)** | `#4BC5FF` | 補助ボタン、メトロノーム、進捗バー。 |
| **Tertiary (Yellow)** | `#FFD64B` | 最高評価(◎)、警告表示。 |
| **Success (Green)** | `#4BFF91` | OK表示、完了。 |
| **Background** | `#F8F9FA` | 基本背景色（薄いグレー）。 |

## 5. 主要な文言 (Wording)
*   **メイン画面**: `リアルタイム練習`, `ビデオ精密評価`
*   **計測中**: `胸骨圧迫を開始してください`, `AI読み込み中`
*   **警告**: `警告：腕を伸ばしてください`, `中断：姿勢を正してください`
*   **ビデオ解析**: `開始3秒と終了3秒を解析対象外とするオートカット機能有効`

## 6. ビデオ解析特有の仕様
*   **解析範囲**: 動画の **[開始3秒後] から [終了3秒前] まで** のみを計測対象とする。
*   **再生挙動**: 動画自体は0秒から最後まで流し、準備時間を視覚的に確認できるようにする。
